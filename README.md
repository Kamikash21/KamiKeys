# KamiKeys Plugin

Este plugin oferece um sistema completo e configurável de gerenciamento de chaves para servidores, com comandos de administração e funcionalidades robustas de banco de dados e personalização.

## Comandos do Jogador

*   `/ativar (key)`: Ativa uma chave de acesso.

## Comandos do Administrador

*   `/gerarkey (tipo) (quantidade)`: Gera uma quantidade de chaves do tipo escolhido.
*   `/darkey (player) (tipo) (quantidade)`: Gera uma chave exclusiva para um jogador específico.
*   `/apagarkey (tipo)`: Apaga todas as chaves ativas de um tipo específico.
*   `/key list (tipo)`: Mostra a lista de chaves ativas (pendentes de resgate).
*   `/key reload`: Recarrega o plugin e suas configurações.

## Funcionalidades do Plugin

*   Geração de chaves aleatórias de acordo com o tipo.
*   Armazenamento das chaves até a ativação, com exclusão automática após o uso.
*   Visualização de chaves ativas para administradores.
*   Geração de log de todas as chaves geradas e resgatadas.
*   Opção de armazenamento via banco de dados (MySQL/SQLite).
*   Plugin totalmente configurável e personalizável com cores.
*   Extremamente completo, configurado, otimizado e organizado.
